Student Grade Management System

Files:
- main.cpp            : C++ source code
- students.csv        : sample data file (CSV)
- USER_MANUAL.txt     : how to run and features
- DESIGN.txt          : proposal, algorithm, pseudocode, function breakdown
- TEST_CASES.txt      : sample test cases and expected outputs

Compile (Windows with g++):
g++ -std=c++17 "main.cpp" -o grade.exe

Run:
./grade.exe

Notes:
- Program automatically attempts to load "students.csv" on start.
- Use menu options to add, update, delete, save or load records.
